# Sea-Level-Predictor

This project analyzes global average sea level data since 1880 and visualizes trends using scatter plots and lines of best fit. It predicts future sea level rise through the year 2050 based on historical data.

## Key Features:

Imports and cleans sea level data from epa-sea-level.csv.

Creates scatter plots to show historical sea level changes.

Uses linear regression to plot lines of best fit for all years and for data from 2000 onward.

Predicts sea level in 2050 based on historical trends.

Visualizations are generated using Matplotlib and regression calculations using SciPy.
